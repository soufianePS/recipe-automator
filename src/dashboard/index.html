              🎨 How to color posted cells in your Google Sheet
              Click to expand
            

            

              

                The Planifier writes posted_at timestamps to columns U / V / W when pins go live.
                To make these visually pop in your sheet, you have 2 options:
              


              

                Option A — Apps Script update (auto-colors on every write, recommended)
              

              
    
                
  1. Open your Apps Script:
                  script.google.com
                  → open the script linked to your sheet (the one in settings.appsScriptUrl).
                
    2. 
                
  2. Replace the entire doPost function with the snippet below.
    3. 
                
  3. Deploy → New deployment → Web app → Execute as: Me, Access: Anyone.
    4. 
                
  4. Copy the new web app URL if it changed, and paste it into your site settings (Settings → General → Apps Script URL).
    5. 
              


              

                
function doPost(e) {
  const body = JSON.parse(e.postData.contents);
  const ss = SpreadsheetApp.openById(body.spreadsheetId);
  const range = ss.getRange(body.range);

  if (body.values) range.setValues(body.values);
  if (body.bgColor) range.setBackground(body.bgColor);

  return ContentService.createTextOutput("OK");
}

                
              


              

                Option B — Conditional Formatting (no code change, 30s setup)
              

              
    
                
  1. Open your Google Sheet → select columns U:W.
    2. 
                
  2. Menu → Format → Conditional formatting.
    3. 
                
  3. "Format cells if…" → Is not empty.
    4. 
                
  4. Fill color:  light green (or whatever you want).
    5. 
                
  5. Save. Done — every cell with a posted_at date now glows green automatically.
    6. 
              

              

                Both options work together — if Apps Script can't color (older script), Conditional Formatting still does the job.
